## 1. Procurement principles

Follows the financier's framework where donor-funded, Cameroon's public procurement code where government-funded; all packages under open competition unless a documented justification applies (the developer-selection question is PRC item 1.3 — it is decided *before* this plan executes, not inside it). Every package below states method, indicative value, and timing keyed to the Master Plan.

## 3. Standing rules

1. **No package proceeds whose upstream data dependency is open** — P2 waits for tier counts (data campaign M8), by design (MIP §7 spine).
2. **Prior review thresholds** set at appraisal; until then all packages ≥ 0.5 bn FCFA treated as prior-review.
3. QR-at-press labelling is **not** a package here — it is a clause in the ministries' existing print contracts (BUD §5.5, Phase 0-H), at publisher cost within regulated pricing.
4. Contract variations above 10% of package value require Steering Committee approval (the CCB, MIP §8) and contingency-line accounting.
5. All awards published (national procurement portal + programme site) — allocation transparency is a programme value the procurement must model.

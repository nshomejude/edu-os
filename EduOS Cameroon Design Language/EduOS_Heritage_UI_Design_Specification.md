# EduOS Heritage UI Design Specification

## Design Philosophy

EduOS combines government authority, academic excellence, African
heritage and modern enterprise software.

## Color Palette

  Token            Hex       Purpose
  ---------------- --------- ---------------
  Heritage Green   #0D5C3B   Brand
  Deep Forest      #063A2A   Sidebar
  Cameroon Gold    #D4A017   Active states
  Ivory White      #FCFBF7   Background
  Emerald          #1E8E5A   Success

## Typography

Primary: Inter

Fallbacks: - Segoe UI - Roboto

Sizes: - Hero 56px - Page Title 36px - Card Title 22px - KPI 36px - Body
16px - Caption 14px - Label 12px

## Component Standards

Cards - Radius 20px - Padding 24px - White background - Soft shadow

Buttons - Height 48px - Radius 14px - Primary green - Secondary white
with green border

Inputs - Height 48px - Radius 12px

Sidebar - Width 280px - Deep Forest background - Gold active state

Top Navigation - Height 88px - Coat of Arms centered - French title
left - English title right

## Heritage Rules

Every page should include subtle national identity: - Circular
university-style EduOS seal - Traditional geometric motifs - Cameroon
map - Gold dividers - Green and gold accents

## Dashboard Layout

1.  Top navigation
2.  Breadcrumb
3.  Page title
4.  KPI cards
5.  Charts
6.  Tables
7.  Quick actions
8.  Footer

## Charts

Allowed: - Donut - Bar - Line - Map

No 3D charts.

## Accessibility

-   WCAG 2.2 AA
-   Keyboard navigation
-   Responsive
-   Mobile-first

## Consistency

Every module shall use the same layout and design language: -
Dashboard - Students - Teachers - Schools - Textbook Distribution -
Inventory - Warehouses - EMIS - Finance - HR - AI - Reports - Settings
